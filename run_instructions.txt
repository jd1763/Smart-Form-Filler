# ----------------------------
# Setup & Run Instructions (User)
# ----------------------------

# 1) Create a virtual environment with Python 3.12
#    • Apple Silicon macOS (Homebrew):
#      /opt/homebrew/bin/python3.12 -m venv .venv
#    • Intel macOS (Homebrew):
#      /usr/local/opt/python@3.12/bin/python3.12 -m venv .venv
#    • Fallback (whatever 'python3' resolves to 3.12):
python3 -m venv .venv

# 2) Activate the virtual environment (every new shell)
source .venv/bin/activate

# 3) Upgrade pip and install runtime dependencies
python -m pip install --upgrade pip
pip install -r requirements.txt

# 4) (Optional) Verify spaCy model is present if the wheel failed for any reason
python -m spacy download en_core_web_sm

# 5) Start the backend API
python -m backend.api

# 6) Load the Chrome extension from ./extension in Developer Mode, open the popup on
a job posting and click "Fill Form"

Note: 
After you do step 1-4 for the first time locally
To run just do the following:
step 4:
    source .venv/bin/activate
step 6:
    python -m backend.api
